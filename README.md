# Retrieval-Augmented Generation (RAG) Pipeline

This project aims to build a question answering system that helps students ask questions and receive answers, while also generating quiz questions based on the NEB curriculum.

It combines curriculum-aligned textbook content retrieval with FAISS-based semantic search and language model generation, delivering context-aware Q&A responses and quiz creation support.

Key aspects:
- Uses NEB curriculum textbook content for retrieval
- Builds a vector index for semantic search
- Generates answers conditioned on retrieved knowledge
- Creates quiz questions from curriculum material
- Supports evaluation of generated responses against reference data
